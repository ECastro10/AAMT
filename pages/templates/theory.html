Mode Formulas (relative to Major Scale)

for example: C, D, E, F, G, A, B = 1, 2, 3, 4, 5, 6, 7
Any changes to that base major formula will be noted with the numbers by b's and #'s
The defining note that makes each mode sound unique looks like this --> [ # ]
