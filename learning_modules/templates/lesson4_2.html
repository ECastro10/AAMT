Note Duration

Okay so you know what a staff and clefs look like but what about the actual notes that go on the staff?
Let's dive into note durations and what they look like, but before we do two things.
One, note duration simply means, how long a note is played or how long you don't play a note.
Secondly, from here on out we start to use simple math such as fractions.
We will begin with a whole note and break it down to smaller bits.

Whole Duration

This is a whole note, that means that it is played for four beats. You will find out more about beats and bars
in the next lesson. All you have to know is that in a bar that goes like this
One - Two - Three - Four
The whole note is played at one and is sustained through the rest of the beats == 4 beat note
Rests are breaks from playing any notes, that means that you would not play any note throughout the count 1-4
NOTE: Whole notes are represented by a hollow notehead. Whole rests are rectangles above any line.

Half Duration

Half notes are half the duration of whole notes
Half rests are half the duration of whole rests.
Two half notes/rests == 1 whole note/rest
NOTE: Notice how the half note simply has a stem added to the notehead and how the the rectangle
is on the bottom vs the top for a half rest. Get used to these simple differences because they mean the world in music.

Quarter Duration

A quarter note is a quarter of the duration of a whole note.
The rest is only a quarter of a whole rest. (Catching on to the pattern yet?)
2 quarter notes/rests == 1 half note/rest
4 quarter notes/rests == 1 whole note/rest
NOTE: The only visual difference in a quarter note is that the notehead is filled.

Eighth Duration

An eighth note is an eighth of a whole note.
The eighth rest is a quarter of a whole rest
2 eighth notes/rests == 1 quarter note/rest
4 eight notes/rests == 1 half note/rest
8 eighth notes/rests == 1 whole note/rest
NOTE: This is where the appearance of a flag begins. A flag is that little flag marking at the opposite
end of the notehead on the stem. Any flag signifies half the value of what the stem and dot portray.

Sixteenth Duration

A sixteenth note is a sixteenth of a whole note
A sixteenth rest is a sixteenth of a whole rest
2 sixteenth notes/rests == 1 eighth note/rest
4 sixteenth notes/rests == 1 quarter note/rest
8 sixteenth notes/rests == 1 half note/rest
16 sixteenth notes/rests == 1 whole note/rest
NOTE: Remember the flag? The addition of a second flag means that the note is half of it's original value
without the newest flag. Prior to the second flag it was an eighth note, half of an eighth is a sixteenth.

Note/Rest Durations on a staff

Notice the new note/rest on right? Can you guess what it is called?
If you guessed thirty second note/rest, you are right.
2 thirty second notes/rests == 1 sixteenth note/rest
4 thirty second notes/rests == 1 eighth note/rest
8 thirty second notes/rests == 1 quarter note/rest
16 thirty second notes/rests == 1 half note/rest
32 thirty second notes/rests == 1 whole note/rest

Dotted Notes

Final concept for note durations.

Dotted notes are unique because you are able to extend the note duration by simply adding a dot.
The rule for dotted notes is : a dot after a note extends the note by half of the notes value.
Think of dots as flags, rather then halving the value it multiplies the value by time and a half.
Examples in image starting at top-left:
Whole dotted note == whole note + half note
Half dotted note == half note + quarter note
Quarter dotted note == quarter note + eighth note
Eighth dotted note == eighth note + sixteenth note

Next part of dotted notes, multiple dotted notes

Before we discuss multiple dots, notice the little lines connecting notes? These are ties.
They basically mean that the note is played for the duration of the notes that are connected.

Last part of dotted notes. Any dot after the first note represents half of the dot before it.
It is easier to understand by explaining the example.
Single dotted half note == half note + quarter note
Double dotted half note == half note + quarter note + eighth note
Triple dotted half note == half note+ quarter note + eighth note + sixteenth note
This can be applied to any note and rest, but do yourself a favor and keep it simple for now.
