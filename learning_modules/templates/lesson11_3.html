Blues Scale

Now we reach the famous Blues Scale.
The beauty of the blues scale is that it is identical to the minor pentatonic scale with one addition.
The blues scale has an added note sometimes called the "blue note". This is simply a flat 5 == b5 or #4.
So in the key of C, our A minor scale == A, C, D, E, G == 1, 3, 4, 5, 7
We add a #4/b5 note to our existing scale which == 1, 3, 4, #4/b5, 5, 7 == A, C, D, Eb, E, G.

FORMULA: W+h, W, h, h, W+h, W

This makes the original pentatonic minor scale into a hexatonic scale or 6 note scale.
Since this scale only adds one note, we will not revisit the staff but instead focus on the pattern.

All the notes except the blue notes should not be new.
So if we are just adding one note, why does it have a new color == blue?
That is because the blue note is only played in passing, it is never a note that you end on because
it does not sound good to end on. So when you practice this pattern try ending on that note and hear
how it just doesn't sound right. Sure you can linger on that note, but you cannot end on it.
Practice this scale as you would all the other ones.
