Aeolian (Minor) Mode

Remember how on the cumulative pattern we have omitted the minor scale extension?
Well this is why, the sixth mode is actually the minor scale.

Aeolian (Minor) Mode

Aeolian Mode: 1 - 2 - b3 - 4 - 5 - [b6] - b7
Unique Note: b6

Since you already know 6th string root pattern, we will skip that one and go straight to
the 5th string root pattern.
Just remember that for the 6th string pattern to visualize the unique note and the mode root.

As you noticed this pattern has no shift but you could easily make one that has a shift.



Remember how the lydian scale didn't extend the cumulative pattern, well the next mode, locrian
is only a semitone away from aeolian so it will not make a difference considering that I put
the aeolian mode before the ionian.
Therefore, here you go, a pattern that stretches 15 frets and has all the modes.
