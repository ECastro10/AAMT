Dorian Mode

Let's take a look at the second mode known as the Dorian Mode.

Dorian Mode

Dorian Mode:
1 - 2 - b3 - 4 - 5 - [6] - b7
Unique Note: 6

Let's discuss this a bit.
In order to achieve the same notes in a key but starting and ending on the second note, we have to first change
the root name and then the mode name.

example: C - D - E - F - G - A - B - C == C Ionian
D - E - F# - G - A - B - C# - D == D Ionian

Notice how at this point the notes are not the same? That is because we are still in the first mode, the intervals
are not appropriate for this layout. We need to flatten following:
3rd & 7th scale degrees
It just so happens that the Dorian mode has this formula.

example: D - E - F# - G - A - B - C# - D == D Ionian
D - E - F - G - A - B - C - D == D Dorian

See how that worked. The basic layout of each mode (or formula, whichever you prefer) allows this new scale
to have different starting and ending points while still maintaining the same notes in the key that you want.

Let's see how it looks like on the fretboard by itself and then with the Ionian Mode.

Key:
Green == Mode root
Red == Regular Root
Yellow == Unique Notes

The rest of the diagrams in the mode chapter will follow this key.
This key is here so that you understand the relationships between the mode, the key, and the unique notes.

So here is the Dorian pattern with the initial root on the 6th string, nothing too crazy.



The shift:
The shift occurs on the 3rd string (G string)
The green note (mode root) should be hit with your pointer finger.


And here is the big pattern if you combine the Ionian and Dorian modes but focus on the key or original root note.



Now that you know two modes, practice them, and focus on those unique notes to hear the differences.
However if you don't want to treat the green notes as home base and hear the green notes, you can treat the red notes as home base
and basically have an expanded range to play the Ionian Mode.
The main takeaway is that we are expanding our knowledge on the fretboard and introducing
new focal points.
