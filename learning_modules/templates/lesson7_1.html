Chord Families

Remember how in the last chapter we discovered that there were only so many major, minor, or diminished chords
per key? When you make chords from the available notes in a major scale, this is known as harmonizing a major scale.
You are free to call these chord families as well because these family patterns are specific to each key.

Triad Chord Families

To recap some notes from last chapter, there are only three major triads, three minor triads, and one diminished triad
per key. So how do achieve this number? Simply create a chord out of every note in that scale. Since there are only
seven notes per major scale, there can only be seven chords. This image illustrate the concept.

Shortcut to remembering chord families, first though, know the notes in that major scale. The pattern for chords
starting from the root note and on:

Major, Minor, Minor, Major, Major, Minor, Diminished.

This pattern works for all keys. Try writing out the chord families for all the keys. You will notice how different
shuffle throughout all the keys.

NOTE: From here on out, we will refer to the chords in each chord family by numbers.
example: I chord == C Major, II Chord == D minor, III chord == E minor, ..., VII chord == B diminished.
