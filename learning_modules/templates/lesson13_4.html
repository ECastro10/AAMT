Building Extended Chords

Chances are that you have heard of other chords besides the chords that we have discussed.
If not, there are 9th chords, 11ths, 13ths, and variations amongst those named.
Once you get past 7th chords, they become extended chords. Although there are non extended chords
that have an extended note added.
example: 6/9 == Major 6 add 9.
Lastly, there are also variations to non extended chords that we did not go over but the formulas
are available on the theory page.

What is an Extended Chord

What makes a chord extended is easier to understand with an example.

Key of C: C, D, E, F, G, A, B
Here you have your notes in the key of C. When it comes to chords we can create, the most
diverse chords that we know only require 4 of these notes inside the same octave.
Remember how in scale degrees C == 1, D == 2, E == 3, etc..?
With one octave you can only have 7 scale degrees hence why you get those 7ths.
To get extended chords, you have to extend that scale past that single octave.

example: C, D, E, F, G, A, B, C, D, E, F, G, A, B
Here we have C == 1, ... B == 7, C == 8, D == 9, E == 10, ... B == 14.
Notice how instead of starting back at one on the C you continue counting?
You are extending that scale. You only go up to the 13th because after that you basically start
repeating again. If this were to happen, you would be playing octaves.
With extended chords, you will notice that some notes become optional which allows you to play
smaller shapes.
I won't dive into how to build them because there are so many extended chords, but there is a
cheat sheet on the theory page. Just scroll down until you get to chord construction formulas.
I recommend exploring these chord types when you are having trouble finding the sound that you want.
