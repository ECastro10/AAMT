Phrygian Mode

Let's jump into the third mode.

Phrygian Mode

Phrygian Mode:
1 - [b2] - b3 - 4 - 5 - b6 - b7
Unique Note: b2

Keeping with the key of C, the phrygian mode allows us to start and stop on the third note
which just happens to be E in the key of C.

example: E - F# - G# - A - B - C# - D# - E == E Ionian
E - F - G - A - B - C - D - E == E Phrygian

And now for the pattern with 6th string root, 5th string root, and combined with the other two modes.

Key:
Green == Mode root
Red == Regular Root
Yellow == Unique Notes



Now for the 5th string root pattern

The shift:
The shift occurs on the 3rd string (G string)
The yellow note (unique note) should be hit with your pointer finger.
