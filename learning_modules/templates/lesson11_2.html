Minor Pentatonic Scale

For the minor pentatonic scale we will be in the key of C, using the A minor scale.
Notes == A, B, C, D, E, F, G == 1, 2, 3, 4, 5, 6, 7
We will remove the same notes that we did in the major scale which were B and F.
The result is: A, C, D, E, G == 1, 3, 4, 5, 7.

FORMULA: W+h, W, W, W+h, W

Next, the two octave pattern. No shifting involved in the pentatonic version of the minor scale.
