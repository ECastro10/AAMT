Intro to Modes/ Ionian Mode (Major)

We briefly touched the concept of modes a while ago when exploring minor scales but we didn't really go into depth.
So let's discuss the concept of modes before diving into our 7 modes.

What is a mode?

In short, a mode dictates the first notes, last note, and the interval spacing.
Let me explain with an example.

example: in the key of C Notes: C - D - E - F - G - A - B - C == C Ionian Mode (first mode) D - E - F - G - A - B - C - D == D "second" Mode
E - F - G - A - B - C - D - E == E "third" Mode
F - G - A - B - C - D - E - F == F "fourth" Mode
...
...
B - C - D - E - F - G - A - B == B "seventh" Mode

Notice how we are keeping the same notes in the key of C but just changing where the scale begins and ends?
This concept is known as modes. I did not include the actual names of the modes because we will learn them
in the order, first, second, third, etc...
There are a few key points that you must understand.

Each mode is unique because the intervals needed to maintain the same notes as the key are different.
example: C - D == 1 whole tone difference
E - F == 1 semitone difference
This shows how the formula for the first mode is different from the third mode.

Because of these unique interval placements, each mode has a unique note that makes it stand out
from the other modes.

It is impossible to stay in the key of C and have C Ionian mode and C second mode or any other mode with the C root.
This is so because of the reason above, each mode has a unique set of intervals. This will become more evident as
you learn more modes.

Lastly, the format for modes are as follows
"Note name" "Mode Name"
example: C Ionian

 Okay, let's go into our first mode, Ionian (Major) Mode.

Ionian Mode (Major)

Great News! You already know the first mode!
How? Well think about it if you start and end on the root note in any key, you end up with a major scale.
It just so happens that the first mode is the major scale. This is why (Major) is next to the Ionian Mode
at the top of the page and this section.
Ionian Mode is the first mode and it is the template that you modify to get all other modes.

Ionian Mode: 1 - 2 - 3 - 4 - 5 - 6 - [7]
Unique Note: 7

That is just about all the information you need to know. As you learn more modes, pay close attention
to that unique note and how the numbers are modified.
Besides that, congratulations, you know the Ionian Mode.
